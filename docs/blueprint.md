# **App Name**: FocusFlow

## Core Features:

- Task Creation and Editing: Create new tasks with titles, descriptions, tags, and priority levels, and edit existing tasks.
- Task Filtering and Sorting: Filter tasks by tags and priority levels, and sort tasks by due date.
- Drag-and-Drop Reordering: Allow users to reorder tasks using drag-and-drop functionality for intuitive organization.
- Focus Mode: Implement an immersive focus mode to minimize distractions while working on tasks.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to evoke a sense of calm and focus.
- Background color: Light gray (#F0F0F0) to provide a clean and unobtrusive backdrop.
- Accent color: A slightly darker shade of blue (#75A4FF) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a clean and modern aesthetic.
- Use simple, minimalist icons for task status, priority levels, and tags.
- Employ a clean, uncluttered layout with ample whitespace to reduce visual noise.
- Incorporate subtle animations for task transitions and confirmations to enhance the user experience.
